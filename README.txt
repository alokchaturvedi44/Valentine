==========================

A minimal, elegant webpage to ask someone to be your Valentine.

Project Structure
-----------------
index.html
your-photo.jpg   (you must add)
music.mp3        (optional)

Setup Steps
-----------
1. Unzip the folder.
2. Add your image and name it exactly: your-photo.jpg
3. (Optional) Add a romantic song and name it: music.mp3
4. Open index.html in any browser.

How to Customize
----------------
Open index.html in VS Code or any editor.

You can change:
- The heading text
- The message paragraphs
- The final thank you line

How to Deploy (so you can send a link)
--------------------------------------
Option 1: GitHub Pages
- Create a repository.
- Upload all files.
- Enable Pages.
- Share the generated link.

Option 2: Vercel
- Import the project.
- Deploy.
- Share the live URL.

Tips
----
Keep the message short, personal, and honest.
Simple words feel more real than dramatic ones.
